Ouvrez une session pour soumettre un rapport ou gérer votre profile ou vos services MaCCG. Si vous avez déjà un compte en ligne avec le gouvernement du Canada, vous pouvez utiliser le même nom d’utilisateur et le même mot de passe pour ouvrir une session.

Si vous avez oublié votre mot de passe ou si vous avez de la difficulté à vous connecter, NE créez PAS de nouvelle CléGC. Si vous ne parvenez pas à récupérer les justificatifs d’identification associés à votre profil MaCCG, veuillez envoyer un courriel à itshelp-aidesti@grainscanada.gc.ca pour obtenir de l’aide.
